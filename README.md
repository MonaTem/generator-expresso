> [Yeoman](http://yeoman.io) generator


## Getting Started

```bash
npm install -g yo
npm install -g generator-expresso
```

Finally, initiate the generator:

```bash
yo expresso
```

## Features

- express
- nunjucks
- gulp
  - autoprefixer
  - sass
  - browser-sync
- mocha/supertest/a11y

# Usage

In development: 

```bash
npm run dev
```

In production: 

```bash
npm start
```
